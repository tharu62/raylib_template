# RAYLIB PROJECT TEMPLATE (WINDOWS)

This repository is pre-configured project for raylib using the raylib 5.5 version for Windows11 with mingw64 AND linux_x64.
The make file is already set up to compile the project without installing raylib locally.

Run this comands to compile and execute a raylib window :

```bash
 make clean && make
```
```bash
 make run
```
